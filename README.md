# XD Library

A representative recreation of an enterprise design-resource hub, built to support the
**XD Library** case study on [christopherrangelux-dev.github.io/Portfolio](https://christopherrangelux-dev.github.io/Portfolio/).

The core idea on display: design resources are organized by how design specialists actually
think — Discipline (UX, Design, Research, Strategy, Content) and Resource Type — rather than a
generic, engineering-driven taxonomy. The demo includes the resource library/browse experience,
a 3-step contribution intake wizard, a manager approval queue, and a taxonomy explorer.

No real company data or branding — mock content only, genericized for public use.

## Running locally

```bash
npm install
npm run dev
```

```bash
npm run build
```
